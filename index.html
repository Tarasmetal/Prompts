Возможности

Всё что нужно для работы с промптами

Мощный набор инструментов для управления текстовыми шаблонами

⚡

Мгновенная вставка

Введите аббревиатуру — и промпт мгновенно вставится в любое активное окно через систему хотстрингов.

📁

Мульти-папки

До 5 папок с CSV-файлами промптов. Включайте и отключайте коллекции чекбоксами в настройках.

🔍

Умный поиск

Мгновенный поиск по названию, аббревиатуре и тексту промпта с подсветкой дубликатов.

🔄

GitHub синхронизация

Скачивайте коллекции промптов напрямую из GitHub-репозиториев одной кнопкой.

🎲

Случайные фразы

Префикс gnr-msg: выбирает случайную фразу из списка при каждой вставке.

💾

CSV формат

Простой и открытый формат хранения: "название","ключ","промпт". Легко редактировать вручную.

📁 Папки с промптами

Укажите до 5 папок, содержащих CSV-файлы. Каждая папка — отдельная коллекция промптов. Чекбокс включает/отключает загрузку папки.

Укажите URL GitHub-репозитория для автоматической загрузки и обновления коллекций промптов.

Настройте хоткеи для открытия окна и вставки промптов. Поддержка раскладко-независимых виртуальных кодов клавиш.

Хотстринги

Вставка по аббревиатуре

Введите сокращение в любом окне — промпт вставится автоматически

Как это работает:

1
Вы вводите
:tr
→
2
Менеджер находит
keys = "tr"
→
3
Вставляется промпт
"Переведи текст..."

Промпт отправляется через буфер обмена в активное окно

🎲 Случайные фразы — gnr-msg:

Если текст промпта начинается с gnr-msg:, при каждой вставке будет выбрана случайная фраза из списка через запятую.

CSV "Приветствие","hi","gnr-msg:Привет!,Здравствуйте!,Добрый день!,Хей!"

Каждый раз при вводе :hi будет вставлена одна случайная фраза из списка.

Синхронизация

Загрузка из GitHub

Скачивайте готовые коллекции промптов из репозиториев

⚙️

1. Укажите репозиторий

В настройках введите URL GitHub-репозитория с CSV-файлами промптов.

🔄

2. Нажмите Update

Скрипт обращается к GitHub API, получает список файлов и скачивает CSV-файлы в указанную папку.

✅

3. Промпты готовы

Коллекция автоматически обновляется, хотстринги перерегистрируются. Можно работать!

Примеры

Как это используют

Два сценария использования Prompts Manager в реальной работе

01

Копирайтер

Ежедневная работа с текстами

1

Создаёт промпты для типовых задач: SEO-тексты, рерайт, заголовки

2

Назначает аббревиатуры: :seo, :rw, :ttl

3

В ChatGPT вводит :seo — промпт вставляется мгновенно

4

Использует gnr-msg: для случайных приветствий клиентам

02

Разработчик

Код-ревью и документация

1

Хранит промпты для код-ревью, рефакторинга, написания тестов

2

Синхронизирует коллекцию команды через GitHub

3

В IDE вводит :cr — получает шаблон для code review

4

Кнопкой Move распределяет промпты по тематическим файлам
